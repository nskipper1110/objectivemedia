ObjectiveMedia

Object-Oriented Multimedia Support

The ObjectiveMedia project seeks to provide object-oriented design and behavior for access to multimedia features such as video/audio capture, playback, encoding and decoding. The goals of this project are as follows: 1. An object-oriented model for multimedia programming. 2. Cross-platform capabilities for Windows, Mac, Linux, Andriod, and iOS. 3. Interoperation for popular Object Oriented Languages such as Java and .Net languages.

Current Feature Set

Windows:

  • Video and Audio Capture
  • Video and Audio Playback
  • H.263+ Video Encoding and Decoding
  • VC-1 Main Profile (WMV9) Decoding
  • G.723.1 Audio Encoding and Decoding
  • Java support through JNI (Java source provided with examples).
